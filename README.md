# 🚫 Página 404 Animada

Página simples de erro **404 Not Found** desenvolvida utilizando **HTML**, **CSS** e **JavaScript** com animações suaves e design moderno.

O projeto foi criado para estudos de interfaces web, animações e experiência do usuário.

---

# 📌 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
